# 2026_03_09_gdp_open_xr_to_measure_pivot
Tool to estimate the central pivot point relative to a controller you want to use.

When you are not using the Quest 3 Oculus SDK or a 3D-printed accessory, the game engine does not know the exact relative reference point.
In Godot, this information is not available.

However, this pivot point is necessary to calibrate my level within the room.

With this tool, you record several key points around a table corner. From these measurements, the system can deduce the relative position of that corner with an accuracy of about 1–5 millimeters.

See [4A calibration project](https://github.com/EloiStree/2026_02_22_godot_nes_a4_and_macro)
